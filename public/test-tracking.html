Analytics Tracking Test Page

Instructions:

  1. Sign in to your analytics dashboard
  2. Add a new site or select an existing one
  3. Click "Install Code" to get your tracking ID
  4. Replace TRACKING_ID_HERE and YOUR_APP_URL in this page's source code
  5. Host this page on any web server
  6. Check your dashboard - you should see real-time tracking data appear!

Important: The tracking code requires:

  • trackingId: Your unique site tracking ID from the dashboard
  • apiUrl: https://snitwwqprnlopkywjmad.supabase.co/functions/v1/track
  • analytics.js: Loaded from your deployed analytics app URL
